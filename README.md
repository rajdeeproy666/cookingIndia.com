# Cooking India 🍲

A simple and beautiful North Indian recipe website built with HTML, CSS, and JavaScript.

## Features

- Home page with cooking-themed photos
- Recipe page with ingredients and cooking steps
- Recipe search by name or ingredient
- Recipe history (recently viewed recipes with localStorage)
- About and Contact sections
- Mobile-friendly responsive menu

## Run locally

1. Clone this repository
2. Open `index.html` directly in your browser

## Deploy to GitHub Pages

1. Go to your repository settings.
2. Open **Pages** in the sidebar.
3. Under **Build and deployment**, choose:
   - Source: **Deploy from a branch**
   - Branch: **main**
   - Folder: **/** (root)
4. Click **Save**.
5. After deployment, your site will be live at:
   - `https://rajdeeproy666.github.io/cookingIndia.com/`
